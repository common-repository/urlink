A link manager for your favorite links.

== Description ==

Manage favorites links easily and group by categories and/or tags. Each links can be set by using direct or redirect and open it in new or current tab.

<h4>Key Features & Options</h4>
<ul>
	<li>Links with categories and tags.</li>
	<li>Links thumbnail.</li>
	<li>Custom redirect/direct.</li>
	<li>Total clicks for each link.</li>
	<li>Custom link relationships.</li>
	<li>Open link target in current or new tab.</li>
	<li>Ascending/descending sort ordering.</li>
	<li>Meta key ordering.</li>
	<li>Display links from specific categories and/or tags.</li>
	<li>Included 3 predifined link list templates.</li>
	<li>Template options.</li>
</ul>

== Frequently Asked Questions ==

= How to use shortcode in the content? =

To use single link shortcode, we can find the shortcode in the link editing page.
For example: <code>[urlink id="2486"]</code>

There is a widget shortcode found in the widget panel. Drag one widget to the Inactive Widgets section,
customize the widget, copy and paste the shortcode into the post/page content.
For example: <code>[urlink-widget id="2"]</code>


= Can we have custom URL for the redirect? =

Yes, for example www.mysite.com/my-custom-redirect/the-link-slug
